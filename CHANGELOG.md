# 更新日志

本文档记录这个维护版分支的重要变化。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，版本号遵循语义化版本思路。

## [v0.1.2] - 2026-04-01

### 新增

- 新增阅读样式设置面板，可直接调整正文宽度、上下左右边距、行间距、字体颜色和高对比模式
- 支持自定义字体颜色，兼容 `#RRGGBB`、`#RGB` 和 `R,G,B`
- 新增基础色模式，适配颜色支持较弱的 Linux 终端

### 改进

- 优化不同终端下的颜色兼容逻辑，支持扩展颜色自动降级
- 修正显示行数逻辑，解决有行间距时 `+ / -` 调整不直观的问题

## [v0.1.1] - 2026-04-01

### 新增

- 新增本地书架首页，支持无参数启动进入书架
- 新增导入本地 `txt` / `epub` 文件与目录批量导入
- 新增导入目录时的当前层 / 递归子目录切换
- 新增拖拽文件或目录到导入输入框获取路径
- 新增正文搜索、结果跳转与当前页高亮
- 新增书签保存、书签列表、删除书签与状态反馈
- 新增阅读内容跨进程缓存，重复打开同一本书更快

### 改进

- 调整阅读区宽度、左侧留白、顶部留白和行间距，提升长文阅读体验
- 阅读界面右侧信息面板改为更适合窄栏的展示方式
- 目录页改成中文提示，并移除了左侧编号
- 书架主列表改为只显示书名，详情信息收敛到右侧面板
- 优化书架分页与列表布局，避免底部内容被遮挡
- 优化从书架打开图书时的加载反馈，避免看起来像卡住
- 优化路径输入补全、候选选择和方向键兼容

### 修复

- 修复 `+` 键在部分终端里无法增加显示行数的问题
- 修复 TXT 书名误从正文首段提取的问题，改为使用文件名
- 修复目录标题和章节标题重复显示的问题
- 修复测试写入真实用户数据目录的问题，改为写入临时目录
- 修复书架标题中常见来源尾巴如 `(Z-Library)`、`_Z_Library` 的清洗问题

## [v0.1.0] - 2026-03-31

### 新增

- 新增 EPUB 阅读支持
- 新增 EPUB 章节目录、章节跳转与目录选择
- 新增阅读进度自动保存与自动恢复
- 新增多行显示与按页滚动
- 新增按终端宽度与中文显示宽度自适应折行
- 新增更偏 IDE / 工作台风格的伪装界面
- 新增多平台 Release 附件
- 新增自动发布 GitHub Release 的工作流

### 改进

- 优化 README 首页说明与下载入口
- 优化发布流程，支持校验文件输出

## 版本说明

- `v0.1.0` 是当前仓库的首个正式版本号发布
